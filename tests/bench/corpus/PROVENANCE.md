# Benchmark corpus — provenance & refresh

How each **representative** fixture was acquired, so it can be re-fetched or replaced.
The **stress** tier is regenerated by `tests/bench/bin/make_corpus.php` and is not listed
here. Licences are read programmatically from the Commons API (`extmetadata`) at fetch
time; see `tests/bench/bin/` history for the fetch helper.

Sourcing policy (see `docs/superpowers/specs/2026-06-06-representative-benchmark-corpus-design.md`,
licence whitelist widened to include **CC BY-SA** by maintainer decision):

- **Photos / generic graphics:** Wikimedia Commons (CC0 / PD / CC-BY / CC-BY-SA).
- **Sprites:** CC0 game-asset sources (Kenney.nl / OpenGameArt CC0).
- **Gaming screenshots:** open-source games (SuperTuxKart / 0 A.D. / OpenTTD); AAA
  screenshots are NOT used.
