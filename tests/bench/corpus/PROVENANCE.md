# Benchmark corpus — provenance & refresh

How each **representative** fixture was acquired, so it can be re-fetched or replaced.
The **stress** tier is regenerated by `tests/bench/bin/make_corpus.php` and is not
listed here.

Sourcing policy (see `docs/superpowers/specs/2026-06-06-representative-benchmark-corpus-design.md`):

- **Photos / generic graphics:** Wikimedia Commons, CC0 / public domain.
- **Sprites & animated sprites:** CC0 game-asset libraries (Kenney.nl, OpenGameArt CC0).
- **Gaming screenshots:** real screenshots of open-source games (0 A.D., SuperTuxKart,
  OpenTTD) — copyrighted AAA screenshots are NOT used.

Every entry must record: source URL, licence, author/attribution, and any normalisation
(resize/convert) applied before committing.
